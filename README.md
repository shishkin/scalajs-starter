# ScalaJS Starter

## Run

Start SBT shell with continuous compilation and webpack server:

    $ sbt
    ...
    sbt:scalajs-starter> fastOptJS
    ...
    sbt:scalajs-starter> fastOptJS::startWebpackDevServer
    ...
    sbt:scalajs-starter> ~fastOptJS
    ...

Open http://localhost:8080 in the browser.
